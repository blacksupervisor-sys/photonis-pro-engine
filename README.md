# 🚀 Mesin Fotonis Pro: Enterprise Edition
**AI-Powered Heavy Equipment Visual Intelligence System**

Sistem otomasi cerdas yang dirancang khusus untuk industri alat berat (**Tatsuo & AIMIX**). Aplikasi ini menggabungkan kekuatan Computer Vision untuk menghapus latar belakang dan Generative AI (Gemini 1.5/2.0) untuk analisa produk otomatis.

## ✨ Fitur Utama
- **Background Removal:** Menghapus latar belakang foto alat berat secara instan untuk kebutuhan katalog.
- **AI Visual Analysis:** Mendeteksi komponen, kapasitas, dan spesifikasi teknis dari gambar.
- **Global Copywriting:** Menghasilkan materi pemasaran metode AIDA dalam Bahasa Indonesia & Inggris.
- **Export to PDF:** Mengunduh hasil analisa langsung ke format PDF siap kirim.

## 🛠️ Tech Stack
- **Frontend:** Streamlit (Python)
- **AI Brain:** Google Gemini API (Multimodal)
- **Image Processing:** Remove.bg API
- **Deployment:** Streamlit Cloud
